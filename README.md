# 河南大学卢福浩课题组软件安装脚本

为进一步简化Linux软件编译及安装方法，特设立此GitHub仓库

一般只需要直接运行程序，即可成功编译安装所需要的生物信息学软件

---

# 准备工作

在你的~/.bashrc里面添加如下一行

export PROGPATH=${HOME}/Programs

其中，${HOME}/Programs为软件安装位置，可自行更改

此仓库的脚本都会默认安装到此目录下
